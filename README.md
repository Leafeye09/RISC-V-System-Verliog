# Single-Cycle RISC-V RV32I Processor (SystemVerilog & Basys-3 FPGA)

##GOAL
The goal of this project was to design a fully functional single-cycle 32-bit RISC-V processor in SytemVerilog, verify it through simulation, and deploy it on a real FPGA hardware. The project was built independently with the help of online research to strengthen my understanding of RTL desgin, computer architecture, XilinixVivado, and FPGA development. These skills are applicable to IC and embedded systems roles in the semiconductor industry. 
## Overview 
This project implements a single-cycle RISC-V RV32I processor from scratch in 
SystemVerilog. The design was verified through automated testbenches and synthesized 
using Vivado 2018.3, achieving timing closure at 91MHz on the Xilinx Artix-7 FPGA.
